1--单元测试？？JUnit用法
  -先pom.xml文件中导入junit.jar包
  -在要测试的方法前面写注解@Test，
  -测试的方法不能是静态方法，即不能用static修饰，
  -下面给出实例
--------------------------------
  package cn.gyyx.test;
	import java.text.SimpleDateFormat;
	import java.util.Date;

	import org.junit.Test;

	public class TestDate {
		@Test
		public void testDate(){
			String date = (new SimpleDateFormat("yyyy-MM-dd")).format(new Date());  
			System.out.println(date);
		}
	}
--------------------------------

4--sqlserver使用
	-向表A添加数据时，要选中表A，然后新建查询，写sql语句，否则会发生找不到表A的情况
	-表名为关键字时，要加""
	例如：表名user，但user为sqlserver的关键字，
	例如，select * from "user" where account='zhaozhu' 
	中，from 后的 "user" 加了""
