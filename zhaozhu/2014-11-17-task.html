--git 练习
1，$ git add '*.txt' ，把所有.txt文件都add到暂存区。
2，$ git diff HEAD   pull之后查看与本地原文件的修改
3，$ git diff --staged 查看暂存区的修改
4，$ git branch clean_up 创建分支clean_up
5, $ git rm '*.txt'
6, $ git push 就可以直接推送
